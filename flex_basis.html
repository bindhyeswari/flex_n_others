Flex Basis

.flex-container { display: flex; width:: 1000px; height: 100px; }
.flex-child { flex-basis: 1000px; }

This defines the default size of an element before the remaining space is distributed. The main-size value makes it match the width or height, depending on which is relevant based on the flex-direction.

Traditional Way

.trad-container { width: 1000px; height: 100px; }
.trad-child { height: 100px; }

Need to specify the container's child height
